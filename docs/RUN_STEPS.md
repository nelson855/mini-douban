# 本轮执行步骤（豆瓣电影标签页）

## 1) 安装依赖
在项目根目录执行：

```bash
npm install
```

执行完成后你要做的事：
- 确认终端没有报错（出现 `added ... packages` / `audited ... packages` 一类提示即可）。
- 然后继续执行第 2 步。

## 2) 启动开发服务器
执行：

```bash
npm run dev
```

执行完成后你要做的事：
- 在终端里找到本地访问地址（通常是 `http://localhost:5173`）。
- 用浏览器打开它，检查页面是否出现：
  - 标题“豆瓣电影”
  - 标签“热门 / 最新 / 高分榜”
  - 点击标签后电影卡片会切换

## 3) 可选：做一次构建检查
如果页面正常，再执行：

```bash
npm run build
```

执行完成后你要做的事：
- 看到 `dist` 产物和 `build` 成功提示即可，说明当前代码可打包。
