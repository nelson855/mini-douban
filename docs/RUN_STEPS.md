## v0.1.0 (2026-03-30)
变更内容：豆瓣电影标签页首次运行与构建检查。

1. 安装依赖：
```bash
npm install
```
2. 启动开发服务器：
```bash
npm run dev
```
3. 可选构建检查：
```bash
npm run build
```
4. 验证要点：
- 页面包含“豆瓣电影”标题。
- 有“热门 / 最新 / 高分榜”标签。
- 点击标签后电影卡片会切换。
